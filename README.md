# gwt-dom-style-definitions
A collection of CSS styles to use with GWT 2 and J2CL.

### Dependency

```xml
<dependency>
    <groupId>org.gwtproject.dom</groupId>
    <artifactId>gwt-dom-style-definitions</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
